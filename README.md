# Object, Reflection and Proxy — Task 2: Destructuring

[![Check Code](https://github.com/Milka79rus/ajs-destructuring/actions/workflows/run.yml/badge.svg)](https://github.com/Milka79rus/ajs-destructuring/actions/workflows/run.yml)

## 📋 Описание проекта
В данном проекте реализована функция `getSpecialAttacks`, которая извлекает массив доступных спец-атак персонажа из сложного объекта. 

Основная цель задания — продемонстрировать владение навыком **деструктуризации объектов и массивов**, а также умение задавать **значения по умолчанию** для потенциально отсутствующих полей.

## 🛠 Технологии
* **JavaScript (ES6+)** — использование метода `.map()` и деструктуризации.
* **Jest** — модульное тестирование.
* **Webpack** — сборка и настройка среды.
* **ESLint** — контроль качества и стиля кода.

## 💡 Логика работы
Функция принимает объект персонажа и возвращает массив объектов атак, каждый из которых содержит:
- `id`
- `name`
- `icon`
- `description`

Если в исходных данных поле `description` отсутствует (атака "засекречена"), функция автоматически подставляет значение: **"Описание недоступно"**.
